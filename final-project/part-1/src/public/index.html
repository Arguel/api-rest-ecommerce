# Method Path Params Description
1 GET http://localhost:8080/products X List the stored products
2 GET http://localhost:8080/products/6136a86ac2131e070ce33848

4 PUT http://localhost:8080/products/6137ad21949a65289b4a1b23
/:id = 6137ad21949a65289b4a1b23
/:id Updates a product. Important: to update the product use the same properties/fields as in the post method (above this)

/:id = 6137850cb8e48912d9d28211
